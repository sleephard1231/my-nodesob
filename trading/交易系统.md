## 📈 双均线交易系统

### 基本原理

使用两条不同周期的移动平均线来捕捉趋势方向：
- **快线：EMA20** — 跟踪短期价格
- **慢线：EMA50** — 识别长期趋势

### 信号规则

#### 🟢 做多信号

- 快线（EMA20）**上穿**慢线（EMA50）→ 金叉
- 价格在均线上方运行
- 均线呈多头排列（快线 > 慢线，且均向上倾斜）

#### 🔴 做空信号

- 快线（EMA20）**下穿**慢线（EMA50）→ 死叉
- 价格在均线下方运行
- 均线呈空头排列（快线 < 慢线，且均向下倾斜）

### 辅助确认

> **均线密集区** = 多周期均线粘合处，是重要的支撑/阻力位。

## 📉 辅助指标

### MACD

- 配合双均线验证趋势强度
- 金叉/死叉与均线信号共振时更可靠
- 关注零轴位置：零轴上方多头主导，下方空头主导

### RSI

- 避免在超买区（>70）做多、超卖区（<30）做空
- 结合均线方向判断背离

# 📋 交易检核清单 — 技术面 + 流动性入场流程

## 第一步：4H 确定大方向

- 均线多头排列 + MACD 零轴上方 → 只做多
- 均线空头排列 + MACD 零轴下方 → 只做空
- 均线缠绕 + MACD 零轴附近 → 震荡，等方向

## 第二步：1H 找入场时机 + 流动性确认

#### 🟢 做多进场信号（4H 多头排列时）

1. 价格回踩 1H 均线，均线走平
2. RSI 回到 30-40 区间
3. MACD 零轴下方准备金叉
4. 成交量：回踩时缩量（卖盘枯竭）
5. 盘口深度：买一挂单增多
6. 价格不创新低，底部抬高

> 全部满足才进场。缺少流动性确认就是假信号。

#### 🔴 做空进场信号（4H 空头排列时）

1. 价格反弹到 1H 均线，被均线压住
2. RSI 反弹到 60-70 区间
3. MACD 反弹到零轴附近准备死叉
4. 成交量：反弹缩量（没有真买盘）
5. 盘口深度：卖盘挂单多，买盘稀疏
6. 反弹高点越来越低

## 第三步：用流动性信号过滤假信号

> **口诀：放量突破收在高，才是真信号；放量突破收在低，小心被骗炮。**

## 第四步：趋势反转三条件

1. 资金费率转正（从负变正，不再一边倒看空）
2. 盘口深度买盘恢复（买一挂单变多）
3. 放量突破前高且收盘站稳

## 交易口诀

```
4H方向做主导，1H入场找信号。
看多做多等缩量，缩量回踩不破底。
看空做空等反弹，反弹无力再进场。
放量突破收在高，才是真信号。
放量突破收在低，小心被骗炮。
```

## 成交量参考

| 场景 | 含义 |
|------|------|
| 放量上涨 + 收在高位 | 真买盘进场 |
| 放量上涨 + 收在低位 | 拉高出货 |
| 放量下跌 + 收在低位 | 真卖盘砸盘 |
| 放量下跌 + 收在高位 | 有人抄底接住 |
| 缩量上涨 | 没人追，可能要跌 |
| 缩量下跌 | 没人卖了，可能要反弹 |

## 亏损的根本原因：真的接盘了还是没有？

- **追涨买盘（市价单）** ＝ 假的，追完就没了
- **深度挂单（挂在那等）** ＝ 真的，说明有人认可这个价格

> 盘口买一侧从稀疏变密集 = 买盘在恢复。买一只有 1-2 个 BTC = 深度很差，别急。
